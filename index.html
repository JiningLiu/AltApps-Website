About AltApps

AltApps is a third-party app store for AltStore, it allows users to install amazing apps for free, directly from the app's website*. It's simple yet modern UI with no ads and iOS 13+ support makes it the best in the world. Beta testers can even install other apps without downloading the IPA**, and choose between the new featured view and list view. It also uses SwiftUI, which means it's super fast and responsive.

*Apps like uYou+ will not be directly from the official download link, because services such as MEGA doesn't support direct download links. IPA files will be downloaded and renamed, then submitted to the AltApps IPA GitHub repo.

**Direct download link of the IPA file of the app is requried, an AltStore direct install link will be generated from the user input.
